# SignalApi

Ruby implementation of the Signal API. API details can be found at [http://dev.signalhq.com](http://dev.signalhq.com)

## Installation

Add this line to your application's Gemfile:

    gem 'signal_api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install signal_api
